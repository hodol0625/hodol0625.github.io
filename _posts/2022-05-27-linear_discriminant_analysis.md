# Linear Discriminant Analysis

참고 : 

LDA는 분류 모델이면서 동시에 차원축소, 데이터시각화 등에 활용되는 알고리즘이다.

LDA의 목표는 PCA와 같이 차원을 축소하는 데, 어떤 차원이냐면 class를 구분하는 정보를 최대한으로 하면서 차원을 축소하는 방법이다.
